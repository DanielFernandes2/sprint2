Nossa proposta visa mudar a forma de comunicação entre a empresa e os médicos, por meio da criação de uma plataforma multifuncional, que oferecerá uma série de ferramentas para engajamento e entendimento mútuo. Solução

O objetivo principal deste projeto é desenvolver uma plataforma exclusiva que substituirá o atual AppTEV, oferecendo um ambiente mais acessível e conveniente para os médicos. A nova plataforma será projetada como um site responsivo, permitindo o acesso por meio de desktops, tablets e smartphones, proporcionando uma experiência do usuário otimizada.

Além da acessibilidade em diferentes dispositivos, a plataforma também oferecerá facilidade na extração de scores e resultados desejados, simplificando o compartilhamento dessas informações quando necessário. Isso tornará a utilização do sistema mais eficiente e acessível, contribuindo para uma melhor compreensão e acompanhamento das informações relevantes.

O escopo do projeto abrange o desenvolvimento da plataforma em HTML/CSS, com o uso de linguagens de programação como Python e JavaScript para garantir sua robustez e escalabilidade. Frameworks como React ou Bootstrap serão utilizados para criar uma interface responsiva e atrativa. A IDE Visual Studio Code será empregada para o desenvolvimento e depuração do código da plataforma.

Integraremos a plataforma a sistemas e serviços externos, como bancos de dados, sistemas de CRM (Customer Relationship Management) e APIs de terceiros. Essa integração permitirá a utilização de dados e recursos existentes, melhorando a eficiência e funcionalidade da plataforma.

Também está previsto o uso de bibliotecas e frameworks de ciência de dados, como pandas, NumPy e scikit-learn, para análise e modelagem dos dados coletados. Algoritmos de aprendizado de máquina, como os disponíveis no TensorFlow ou PyTorch, serão implementados para personalização e recomendação na plataforma.

Além disso, visamos introduzir elementos de gamificação na plataforma. Por meio de bibliotecas e frameworks específicos, como pontos, recompensas e desafios, pretendemos criar uma experiência envolvente e divertida para os usuários.

O porque fazer essa solução

A implementação dessa nova plataforma é fundamental para a Sanofi estabelecer uma relação mais próxima e produtiva com os médicos prescritores. Atualmente, a comunicação é limitada e pouco eficaz, dificultando o engajamento e a compreensão mútua.

A proposta de uma plataforma multifuncional visa suprir essas lacunas, fornecendo aos médicos uma experiência aprimorada no uso da plataforma, facilitando o acesso aos recursos disponíveis e incentivando uma participação mais ativa por parte deles.

Com a introdução de elementos de gamificação, esperamos estimular a interação dos médicos com a plataforma, tornando a experiência mais envolvente e recompensadora. Isso contribuirá para o engajamento contínuo e a fidelização dos profissionais à plataforma.
